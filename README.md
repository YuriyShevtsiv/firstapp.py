# Мій перший GUI-додаток Tkinter

Цей додаток є простим прикладом створення графічного інтерфейсу за допомогою бібліотеки Tkinter в Python. Додаток містить вікно з текстовим полем для вводу та кнопкою, яка змінює вміст елемента тексту (лейбла) на основі введеного тексту.

## Опис додатку

Цей графічний інтерфейс складається з двох основних частин:
1. **Основний лейбл (Label):** Показує текст, який змінюється в залежності від вводу користувача.
2. **Поле вводу (Entry) та кнопка:** Дозволяє користувачеві ввести текст і натискання кнопки змінює вміст лейбла.

### Основні функції:
- **Оновлення тексту:** Користувач може ввести текст у поле вводу. Після натискання кнопки, лейбл оновлюється і відображає введений текст.
- **Зміна кольору тексту:** Текст на лейблі змінює свій колір на жовтий, коли користувач натискає кнопку.

## Як запустити

1. Переконайтесь, що у вас встановлений Python та бібліотека Tkinter. Tkinter зазвичай входить у стандартну бібліотеку Python, тому додаткові інсталяції можуть не знадобитись.

2. Скопіюйте код у файл з розширенням `.py`, наприклад `first_gui_app.py`.

3. Запустіть файл за допомогою Python:

   ```
   python first_gui_app.py
   ```

Після запуску програми ви побачите вікно, де можна ввести текст, і при натисканні кнопки буде оновлюватись лейбл.
